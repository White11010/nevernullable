# nevernullable

`Option<T>`, `Some<T>`, `None` types and runtime functions to safely write JS/TS code without using `null` and `undefined`

## Installation

```
$ npm install nevernullable
```

## Usage

-  [Importing](#importing)
-  [Option](#option)

## Importing

```
import { Option, Some, None } from 'nevernullable';
```
